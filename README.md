RankPlugin lets you display player ranks as PNG icons from a resource pack in both chat and the tab list. Ranks are defined in ranks.yml, can be assigned via command, or selected through a GUI.

✨ Features

📌 Show rank icons in chat

📌 Show ranks in the tab list

📌 /setrank <player> <rank> for quick rank assignment

📌 /rankgui for GUI-based rank selection

📌 Easy customization through ranks.yml

📌 Full resource pack support with PNG icons

⚡ Installation

Put the plugin .jar into your plugins folder.

Restart the server.

Edit plugins/RankPlugin/ranks.yml to configure your ranks.

Add PNG icons for each rank in your resource pack.

Test in-game using /setrank or /rankgui.
